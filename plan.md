- 아기 이유식 레시피를 공유할 페이지를 만듦
- 링크를 가지고 있는 다수가 접근 가능한 웹페이지
- github pages를 사용해서 도메인 주소를 할당해줘 (github pages를 사용해본 적 없는 상태로, 추후 가이드해줘)
- 전달할 레시피는 URL 형태로 제공하고 그 url은 영상을 포함하고 있음.
- 영상 아래 텍스트 형태로 레시피를 보여줄 것 - 재료, 조리과정
- 레시피 모음 폴더 하위에 요리를 추가할 예정이고, 게시글 형태로 요리가 저장되었으면 함

## 결정사항
- 레시피 원본 관리 형식: Markdown(.md)
- 페이지 구조: 목록 1개 + 레시피 상세 여러 개
- 영상 출처: YouTube 임베드
- 배포 방식: 순수 HTML/CSS 정적 사이트 + GitHub Pages
- 공개 범위: 링크 공유 중심 공개 운영(기술적으로는 Public)

## 구현 전 확정 규칙
- 레시피 필수 항목: 제목, slug, YouTube URL, 재료, 조리과정
- URL/파일명 규칙: slug는 영문 소문자 + 하이픈(-) 사용
- 상세 페이지 레이아웃: 재료/조리과정 텍스트 상단, 영상 하단
- 사이트 기본 구조: index.html(목록) + 레시피별 상세 페이지

## 운영/배포 정책
- GitHub Pages 초심자 기준으로 저장소 생성부터 배포까지 단계별 가이드 필요
- 검색 노출 최소화 방향으로 운영(완전 비공개는 불가)
- 영상 비공개/삭제, 잘못된 링크(404) 등 예외 상황 안내 문구 준비

## 제외 범위
- 로그인/회원 기능
- 댓글/좋아요
- 서버 API/DB
